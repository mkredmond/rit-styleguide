## There are multiple ways to create a link with an icon. Here are a few examples:

**Note** that the icon color is derived from the `--fa-color` css variable. The CSS rules should automatically set this corrects for the various background colors (`.bg-black`, `.bg-light`, `.bg-primary`) but it can be overridden in the situation it needs to be. This can be done via a class redeclaring it's value or simply adding `style="--fa-color: pink"` to the tag itself.

Since these are FontAwesome classes, you can use `fal`, `far`, or `fas` to set the icon weight. The default is `far` (regular).
